## Why Zoho Creator is SaaS

Zoho Creator is a Software as a Service (SaaS) platform because it allows users to create and manage applications through a web browser without installing software. The applications, databases, and services are hosted on Zoho's cloud infrastructure. Zoho manages software updates, server maintenance, security, backups, and availability, while users focus only on designing and using their applications.

---

## Cloud Layer Mapping

| Layer | Responsibility |
|-------|----------------|
| IaaS | Provides servers, storage, and networking infrastructure. |
| PaaS | Provides the application development environment. |
| SaaS | Provides ready-to-use cloud applications such as Zoho Creator. |
